# Thứ tự triển khai v2.4.0

1. Giải nén `friendzones-r2-media-v1.zip`.
2. Chạy `upload-r2.bat`, điền Access Key/Secret vào `.env.r2` trên máy cá nhân và upload media.
3. Kiểm tra bốn URL hero trong hướng dẫn.
4. Deploy ZIP website v2.4.0 lên GitHub/Vercel.

Không đưa `.env.r2`, Access Key hoặc Secret Access Key lên GitHub.
